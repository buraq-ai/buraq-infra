# ADR-006: Vector DB Selection — Deferred Pending Technical Spike

## Status
Pending

## Context
The RAG pipeline requires a Vector Database to store and
retrieve document embeddings. Multiple options exist including
Pinecone, Weaviate, Chroma, and pgvector.

## Decision
Decision is deferred until a technical spike is completed
to evaluate performance, cost, and self-hosting capabilities
of the available options.

## Consequences
- A spike must be added to the backlog immediately
- Spike owner is the AI Developer
- Decision must be made before Sprint 2 begins
- Temporary solution may use an in-memory store for prototyping
