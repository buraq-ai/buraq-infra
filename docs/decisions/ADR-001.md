# ADR-001: Microservice Architecture (Java + Python Separated)

## Status
Accepted

## Context
The system requires both enterprise-grade backend capabilities
and AI/ML processing. These have different technology requirements,
scaling needs, and development lifecycles.

## Decision
We will separate the Java Spring Boot backend and the Python
FastAPI AI service into independent microservices with
their own repositories and deployment pipelines.

## Consequences
- Each service can be scaled independently
- Teams can work in parallel without conflicts
- Requires strict API contracts between services
- Adds complexity to deployment and monitoring
