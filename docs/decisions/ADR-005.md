# ADR-005: Angular + Tailwind CSS for Frontend

## Status
Accepted

## Context
The system needs a modern, maintainable frontend framework
that supports enterprise-grade applications with
multilingual support (Arabic, French, English).

## Decision
We will use Angular as the frontend framework combined
with Tailwind CSS for styling.

## Consequences
- Team must be familiar with TypeScript and Angular
- Tailwind CSS removes the need for custom CSS files
- Angular's built-in i18n module will handle multilingual support
- Bundle size must be monitored and optimized
